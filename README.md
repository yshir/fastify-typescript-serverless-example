# fastify-typescript-serverless-template

## Description

- [fastify](https://www.fastify.io/)
- [TypeScript](https://www.typescriptlang.org/)
- [AWS Lambda](https://aws.amazon.com/lambda/)

## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
$ npm run dev
```

## Running the test

```bash
$ npm test
```
