# fastify-typescript-serverless-example

## Description

- [fastify](https://www.fastify.io/)
- [TypeScript](https://www.typescriptlang.org/)
- [AWS Lambda](https://aws.amazon.com/lambda/)

## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
$ npm run dev

# production mode
$ npm run start
```
